# Contributing

Thanks for helping make this resource better! 🙌

## How to contribute

1. **Fork** this repository
2. Create a branch: `git checkout -b add-new-question`
3. Add your question(s) following the existing format:
   - `### Q<number>. <Question>` heading
   - `**Answer:**` with a clear, concise explanation
   - Code examples in ```java fenced blocks where helpful
4. Open a **Pull Request** with a short description

## Guidelines

- Keep answers **short and interview-ready** — no walls of text
- Verify code compiles and is correct
- Add questions to the file matching their topic and difficulty
- Fixing typos / improving explanations is equally welcome!
